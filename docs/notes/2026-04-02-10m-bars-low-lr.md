# 10m bars from scratch with LR=5e-6

**Date:** 2026-04-02
**Status:** Completed

## Setup

- Trained from base Gemma 3 270M (not from checkpoint)
- 10m bars, SIGMA_WIN=144, all 748 symbols
- Data: v3, 1.84M train / 229K val / 229K test sequences
- LR=5e-6 (reduced from 3e-5), BATCH_SIZE=8
- 1 hour training, 9990 steps

## Results

- val_loss: 2.623 (worse than v2's 1.985 — model didn't converge enough)
- train_loss: 2.627
- Loss was still declining at end — LR=5e-6 needs more training time
- Directional accuracy: 49.94% (random)
- No context length effect: all bar position ranges ~50%

## Context length analysis

## Takeaway

- LR=5e-6 too low for 1-hour training from scratch — loss still declining at 2.6
- Despite lower loss in v2 (1.98 vs 2.62), directional accuracy was the same (~50%)
- This suggests the model learns token distribution well but not directional signal
- Lower loss ≠ better directional prediction with this tokenization
